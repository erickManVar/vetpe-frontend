# vetpe-frontend

Frontend del marketplace veterinario vetpe.com — vets a domicilio verificados.

## Stack
- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- better-auth
- Culqi (pagos escrow)

## Features
- Búsqueda de vets por distrito y especie
- Agendar visita a domicilio
- Perfil de mascotas del cliente
- Historial médico

## Dev
```bash
npm install
npm run dev
```
